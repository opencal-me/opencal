# opencal

_A lowkey way to share what you're up to as an open invite._

## Setup

```bash
# Install tools
brew install docker rbenv nodenv yarn watchman overmind

# Set up environment
git clone git@github.com:opencal-me/opencal
cd opencal
bin/setup
```
